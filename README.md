# Онлайн-библиотека

Cкрипт генерирует статические HTML‑страницы онлайн‑библиотеки из файла meta_data.json и шаблона template.html. 
Результат — папка pages/ с файлами index.html (первая страница), index1.html, index2.html и т.д.

## "Проект"

Проект опубликован на GitHub Pages - [Online_library](https://himas1321.github.io/online_library/), достаточно просто перейти по ссылке на сайт и читать книги онлайн.

## "Запуск"

- Скачайте код
- Установите зависимости командой `pip install -r requirements.txt`
- Запустите livereload-сервер командой `python render_website.py`

После запуска render_website.py перейти по [ссылке](http://127.0.0.1:5500).


## Цели проекта

Код написан в учебных целях — это урок в курсе по Python и веб-разработке на сайте [Devman](https://dvmn.org).
